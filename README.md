# gwj 91: gran theft aubrey

- granny that steals purses
- all persons are granny

## Mechanics
- rock paper scissors 
- pull hold push

- pull beats hold - purse yanked
- hold beats push - pusher slips
- push beats pull - puller falls down
- every move causes arthritis to worsen
- must beat all grannies with one move to beat the game (???)
